Privacy Policy

Effective Date: January 2026

LR SMS Forward we respects your privacy. This Privacy Policy explains how our mobile application, LR SMS Forward, handles user information.

Information Collection

LR SMS Forward does not collect, store, or request any personal or sensitive user information. The application does not require user registration, login, or account creation.

Use of Information

The app is designed solely to forward SMS messages based on user configuration. No user data is processed, saved, or transmitted to external servers.

Data Sharing

We do not share, sell, rent, or disclose any user data to third parties. The application does not use analytics services, advertising networks, or tracking tools.

Permissions

LR SMS Forward requests access to SMS messages only to perform its core functionality. This permission is strictly used for forwarding SMS messages as configured by the user and is not used for any other purpose.

Data Security

Since the application does not collect or store user data, there is no risk of data leakage or unauthorized access.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated effective date.

Contact Information

If you have any questions or concerns regarding this Privacy Policy, you may contact the developer through the GitHub repository associated with this project.
